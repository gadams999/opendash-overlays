# Requirements

## System Requirements

| Requirement | Details |
|---|---|
| OS | Windows 10 (1903+) or Windows 11 |
| Discord | Discord desktop app (stable or PTB), running and signed in |
| .NET | .NET 10 runtime (bundled in MSI installer) |
| Architecture | x64 only |

See [Common Setup](/common-setup/) for the full operating system setup guide shared across all OpenDash overlays.

## Software Prerequisites

- Discord desktop app installed, running, and signed in to an account
